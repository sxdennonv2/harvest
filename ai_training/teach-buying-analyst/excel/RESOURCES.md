# Excel to a High Standard — Resources

Curated, high-trust sources. Knowledge for lessons is drawn from here, not from guesses.

> Note: web access was unavailable when this file was seeded, so links are the canonical,
> long-stable URLs for each source. Verify/trim on next online session.

## Knowledge

- [ExcelJet — Dave Bruns](https://exceljet.net)
  Clean, example-first reference for functions, formulas and shortcuts. Use for: precise
  function syntax, reference behaviour, keyboard shortcuts, dynamic-array patterns.
- [Microsoft Support — Excel help & learning](https://support.microsoft.com/excel)
  Authoritative primary source for feature behaviour and version differences. Use for:
  official function docs, Tables, dynamic arrays, "what changed" between versions.
- [Microsoft — Excel functions (by category)](https://support.microsoft.com/en-us/office/excel-functions-by-category-5f91f4e9-7b42-46d2-9bd1-63f26a86c0eb)
  The canonical function index. Use for: discovering the right modern function (XLOOKUP,
  FILTER, SORT, UNIQUE, LET, etc.).
- [Chandoo.org](https://chandoo.org)
  Practitioner blog aimed at analysts and dashboards. Use for: reporting layouts, analytical
  patterns, "how a pro structures a model".
- [Excel Campus — Jon Acampora](https://www.excelcampus.com)
  Strong on productivity, keyboard technique and Tables. Use for: undoing slow-mouse habits.

## Wisdom (Communities)

- [r/excel](https://www.reddit.com/r/excel/)
  High-signal, well-moderated. Use for: "is there a better way to do X", formula critique.
- [MrExcel Message Board](https://www.mrexcel.com/board/)
  Deep archive + active experts. Use for: gnarly real-world problems, second opinions.
