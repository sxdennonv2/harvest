The F4 cycle

A1 → $A$1 → A$1 → $A1 → back to A1. Select the reference in the formula bar first.

Read it in one breath

The $ locks whatever comes right after it. $A1 locks the column; A$1 locks the row.

Smell test If you're retyping the same value into many rows, or fixing drift by hand after a fill — you needed a $.
