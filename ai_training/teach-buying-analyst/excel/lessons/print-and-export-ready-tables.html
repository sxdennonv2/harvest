Excel to a High Standard · Lesson 0017

Print- & export-ready tables

Tied to your mission: the analysis is right, but it prints across four ragged pages with the headers stranded on page one. This lesson is the last mile — page setup that turns a working sheet into a clean, one-page report your director can read on paper or as a PDF.

Practice workbook · open this first

Open the course workbook and click the Print ready tab — a buying report with a dozen rows is set up. It looks fine on screen; your job is to make it print clean.

The one idea

A worksheet and a printed page are two different things, and Excel gives you four controls to reconcile them: a print area (print only what matters), orientation and scaling (Landscape, Fit to one page wide), print titles (repeat the header row on every page), and headers/footers (a page number, a report name). Set them once and every print or PDF comes out tidy.

Why it matters to a buyer

A report is judged the moment it lands — and a sheet that spills over pages with no repeating header reads as careless, whatever the analysis says. Clean output is respect for the reader's time: they see the whole picture on one page, with headers where they expect them. It's also how a workbook becomes a shareable PDF instead of a file only you can navigate.

The bad habit this undoes Hitting Print (or Save as PDF) straight from the working sheet and shipping whatever falls out — columns cut off, headers on page one only, a stray helper column in the middle. Two minutes in Page Layout is the difference between "a report" and "a mess someone has to decode".

How to do it

On the Print ready tab: select the report range and set Page Layout → Print Area → Set Print Area. Choose Orientation → Landscape, then Scale to Fit → Width: 1 page. Open Page Layout → Print Titles → Rows to repeat at top and pick row 1 so the header appears on every page. Finally, in the same dialog's Header/Footer tab, add a footer with the page number.

Always preview: File → Print (or Ctrl+P) shows the exact pages. Fix it here, not after it's printed.

Try it for real

Set the print area, orientation, scaling, repeating header and footer on the Print ready tab, then press Ctrl+P and read the preview: one page wide, header on top, page number below. Then File → Export → Create PDF and open it — that PDF is what you'd actually send. If anything's cut off, back to Page Layout.
