Excel to a High Standard · Lesson 0019

Named ranges

Tied to your mission: a formula that reads =SUM(Revenue) tells the next person exactly what it does; =SUM($F$2:$F$19) makes them go and look. A named range gives a cell or block a plain-English name you can use anywhere in the workbook — clearer formulas, fewer mistakes, and one place to update when the data grows.

Practice workbook · open this first

Open the course workbook and click the Named ranges tab — a short product-and-revenue list sits ready for you to name the revenue column and total it by name.

The one idea

Select a range, type a name in the Name Box (the little box to the left of the formula bar), press Enter — that range now answers to the name. Use it in any formula: =SUM(Revenue), =AVERAGE(Revenue), =MAX(Revenue). Names are absolute by nature, so they never slide when you copy a formula.

Manage them all under Formulas > Name Manager — rename, delete, or repoint a name at a bigger range in one place, and every formula that uses it updates. Rules: names can't contain spaces (use Net_Revenue) and can't look like a cell reference. A name can also point at a single cell — perfect for a rate or threshold you reuse.

Why it matters to a buyer

A workbook full of =SUM($F$2:$F$19) is a workbook only its author can safely change. Names turn formulas into something a colleague can read and trust at a glance, and they remove a whole class of error — you can't accidentally point at the wrong column when the column has a name. When the data grows, you repoint the name once instead of hunting every formula.

The bad habit this undoes Scattering raw $F$2:$F$19 ranges through dozens of formulas, then breaking half of them when the data grows by a row and you update only some. Name the range once and point everything at the name.

How to do it

On the Named ranges tab, select the revenue figures in B2:B11, click into the Name Box, type Revenue and press Enter. Now in D2 type =SUM(Revenue) and in D3 type =AVERAGE(Revenue) — both read clearly and total the named block. Open Formulas > Name Manager to see Revenue listed; that's where you'd extend it if the list grew.

Builds on references: a name is really an absolute reference with a friendly label — the same "lock it so it doesn't move" idea from the references lesson, made readable.
