Excel to a High Standard · Lesson 0003

Data validation: controlled dropdowns

Tied to your mission: when you (or a colleague) type a category by hand, one week it's "Snacks" and the next it's "snack" or "Snaks" — and every SUMIFS and pivot silently splits in two. Data validation stops the typo at the door: a cell that only accepts values from an approved list, chosen from a dropdown instead of typed.

Practice workbook · open this first

Open the course workbook and click the Data validation tab — an order-entry block is waiting, with the approved Category and Supplier lists set out to the side.

The one idea

Select the cells that should be constrained, go to Data ▸ Data Validation, choose Allow: List, and point Source at the range of approved values. Each cell now shows a dropdown arrow and rejects anything not on the list. One setup protects every row you'll ever type into that column.

Point the source at a proper list (better still, a Table column or a named range) and the dropdown grows automatically as you add approved values — no need to reopen the dialog.

Why it matters to a buyer

Every downstream total depends on categories and supplier names being spelt identically. A single stray spelling fractures a SUMIFS total and makes a pivot show two rows where there should be one. Validation makes clean input the default, so your analysis can trust its own keys.

The bad habit this undoes Free-typing categories and suppliers, then hunting down "Coastal Co" vs "Coastal Co." vs "coastal co" after the totals look wrong. Fix the input and the outputs stop lying.

How to do it

On the Data validation tab, select B2:B11 (the Category column of the order block), open Data ▸ Data Validation, set Allow: List and Source: =$H$2:$H$5 (the approved categories). Do the same for C2:C11 against the supplier list in $I$2:$I$7. Now click a cell in column B — a dropdown appears, and typing "Snaks" is refused.

Builds on Tables: if your approved list is a Table column, use it as the source; the dropdown extends itself every time you add a new approved value.
