# Start here

Welcome — this is a self-contained Excel learning workspace.

## Just want to learn (no setup)
Open **`lessons/relative-and-absolute-references.html`** in your browser and work through
it. Every lesson is a single self-contained page with a built-in self-check. Reference sheets
live in `reference/`. See the whole plan in **`CURRICULUM.md`** (theme × capability-ladder grid).

## Want the full guided experience (with the `teach` skill in VS Code + Copilot)
1. Open this folder in VS Code.
2. Edit **`MISSION.md`** so the *Why* is in your own words — this is what tunes lessons to you.
3. Invoke the `teach` skill and point it at this folder. It will read your mission, ask a few
   calibration questions, and author your next lesson at the right level.

## What's personal vs shared
- `lessons/`, `reference/`, `assets/` — shared teaching material, same for everyone.
- `MISSION.md`, `NOTES.md`, `GLOSSARY.md`, `learning-records/` — **yours**. They track your
  progress. Don't copy someone else's; start these fresh (this starter already has them blank).
